# Descrição do Projeto ADSI

O ADSI — Assistente Digital Simplificado é um serviço digital de inclusão tecnológica voltado para pessoas com dificuldade no uso de dispositivos digitais, especialmente idosos e iniciantes.

A proposta do sistema é oferecer uma experiência adaptada ao dispositivo mais utilizado pelo usuário. Ao acessar a plataforma, o usuário escolhe entre Notebook/Computador, Celular ou Ambos.

Caso selecione Notebook/Computador, será direcionado para uma interface web com tutoriais passo a passo sobre tarefas básicas, como ligar e desligar o notebook, acessar a internet, enviar e-mails, salvar arquivos e se proteger de golpes digitais.

Caso selecione Celular, será direcionado para atendimento via WhatsApp, onde poderá receber orientações simples por meio de um chatbot. Essa escolha facilita o uso para pessoas que já estão familiarizadas com o WhatsApp.

A solução busca promover autonomia digital, acessibilidade e segurança no uso da tecnologia.
